Сюда нужно положить PNG-портреты для модалок.

Точные имена файлов:
- barracks.png
- lavka.png
- workshop.png
- mercenaries.png
- king.png
- mage.png
- master.png

Эти файлы используются так:
- barracks.png -> Казарма
- lavka.png -> Лавка
- workshop.png -> Мастерская
- mercenaries.png -> Наемники
- king.png -> Король / Задания города
- mage.png -> Маг
- master.png -> Великий мастер
